> 玩崇祯，目标 **100 个月内**把开局三局势全部推到 good 端（bar→100），且不靠透支内库硬撑。
> 主线三招：**开源（开海+工厂科技）/ 招募（科技人才）/ 提国库（查贪查隐田+打阉党提皇威）**。

## 三、三招推演（100 月分三阶段）

### 阶段 1：1–12 月　立威 + 起底（皇威 20→45，止血）

主线 = **打阉党提皇威**，皇威是后面所有增税的前置。

1. **诛阉党余孽 / 清内廷**：起复东林清流，皇威 +。开局已诛魏忠贤，继续清算崔呈秀党羽。
2. **起复徐光启**（礼部/工部），挂"西学农政"差事 → 为阶段 2 工厂铺路。
3. **起复袁可立**督登莱 → 稳辽东海路 + 开海前哨。
4. **查贪起手**：先查一两个阉党贪官抄家，**抄没银直入内库/国库**（一次性回血 + 立威双收）。
5. 辽东先**保底拨饷**压住"辽东索饷"别跌穿（用内库缓冲，别让 bar→0 哗变）。

阶段末目标：皇威≥45，户部亏空 bar 止跌回 35+，三局势无一跌穿 15。

### 阶段 2：13–48 月　开源（户部亏空 35→70，国库转正）

皇威够了才动税基。**财政三板斧同时推**：

1. **查隐田 + 清丈**（田赋率 68→85）：徐光启/毕自严主持，分省推。最大单笔 +25/月，但最招士绅恨 → 靠阶段1攒的皇威扛。
2. **开海设关**：诏书起复袁可立督登莱、**下旨招抚郑芝龙**（海盗反正→授水师官），开月港+广州市舶，关税入国库 +10~15/月。
3. **工厂科技**（徐光启主理）：
   - 火器局/铸炮厂扩产 → 辽东边军换装（同时喂"辽东索饷"good 端）
   - 玻璃/肥皂工厂 → 商品入国库；化肥 → 喂 regions grain_output → 间接解陕西
4. **削藩减支**（宗室禄米率 55→30）：省 30/月，但撞宗室，放阶段2后半、皇威更高时推。

阶段末目标：国库月净转正（>0），户部亏空 bar≥70，国库存量回升过 350。

### 阶段 3：49–100 月　解辽东 + 陕西（两 bar→100）

财政转正后，钱砸向边防与赈济：

1. **辽东**：足饷 + 火器换装 + 袁可立海路补给 → 宁锦稳固，辽东索饷 bar→100。
   - 关键：欠饷清零（arrears→0），morale/training 拉起。
2. **陕西**：化肥提粮 + 开仓赈灾 + 招抚饥民（就抚而非纯剿）→ 流寇 bar→100。
   - 钱够才赈得起；阶段2不转正这步无解。
3. 三 bar 全 good → 通关。皇威/民心顺势抬到 70+。

## 四、关键顺序铁律（错序必崩）

1. **皇威先于增税**：皇威<40 硬查隐田/削藩 → 士绅宗室反扑，民心崩、政令打折。
2. **财政先于边防/赈灾**：国库没转正就猛拨辽东赈陕西 → 内库透支，连锁崩。
3. **赈抚先于剿**：陕西纯剿不赈 → 流寇 bar 反跌（越剿越多）。
4. **内库是缓冲不是钱包**：只在 bar 将跌穿时一次性救急，别当常规收入。
